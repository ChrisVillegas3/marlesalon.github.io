¿Cómo recuperar mi contraseña?

¿Que código ingresar?

¿Cómo actualizar mi nueva contraseña?
Ingrese una nueva contraseña y reescribala para confirmar.

La contraseña debe de contener como mínimo una letra mayúscula, una letra minúscula, un número y un caracter especial.
